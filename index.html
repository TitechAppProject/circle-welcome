もっと便利に
Titech App Project は、アプリや web サービスの提供を通して
東工大生の学生生活をよりよくすることを目指すデベロッパーサークルです。
興味がある方はこちら
主なサービス
TitechApp
詳しくみる
大学のポータルサイトや学内WiFiに簡単にアクセスできるアプリです。iOS / iPadOS / macOS / Android 向けにそれぞれ展開。
Titech Info
詳しくみる
東工大の情報を集めたポータルサイトです。公認・非公認を問わず学内のサークルの新歓情報などを掲載しています。
Titech App Projectでできること
学生に幅広く利用されているプロダクトを通じて、企業で行われているようなプロダクトリリースを経験することができます。1つのポジションを極めるだけでなく、自分の中にあるアイデアを他の人と協力して実現させたい人を歓迎します。
エンジニア
iOS, Android or Web Engineer
プロダクトの実装を担当します。iOS や Web を中心に、Android やデータベースなど幅広い領域を学ぶことができます。 「TitechApp にほしい機能を自分で作りたい」「トレンドの技術を実際に試す場がほしい」など、参加の動機も人によって様々です。
デザイナー
UX and UI Design Engineer
サークルが提供するすべてのプロダクトのデザインを担当します。その領域はユーザの目に触れる UI だけに留まらず、企画やリサーチ・UX 設計にも及びます。デザインの勉強だけでなく新機能や新サービスをゼロから提案してみたい方もお待ちしています。
企画・PM
Planner and Product Manager
新しい機能やサービスを提案・要件定義したり、プロダクトに関わるメンバーのコミュニケーションを円滑にしたりする役割を担います。 特に PM はなかなか馴染みのない概念ですが、複数の人が関わるプロダクトのリリースを実現するためには必要不可欠な存在です。
TitechApp の開発に参加しませんか？
プロダクトリリースに興味はあるけど、なにをやりたいのか決まっていない……そんな状態の方も、一度せひ参加してみてください。
